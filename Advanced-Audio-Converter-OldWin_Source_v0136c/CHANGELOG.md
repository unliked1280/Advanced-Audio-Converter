# Changelog

## 0.13.6c pre-stable

- Manual dark theme fallback default changed to OFF for clean installs.
- Follow Windows theme remains ON by default.
- Use Windows accent color remains ON by default.
- System information output polished:
  - `GenuineIntel` / `AuthenticAMD` vendor prefixes removed from normal display.
  - `Intel(R)` / `Core(TM)` / extra `CPU` text cleaned.
  - `Micro-Star International Co., Ltd.` shortened to `MSI`.
  - RAM summary formatting made shorter and cleaner.
- Build-script typo fixed in fixed package.

## 0.13.x

- Added System information tab/log output.
- Added FFmpeg thread selection up to OldWin cap of 4.
- Added Windows long-path helper window.
- Removed compact interface mode in favor of auto-scale to current screen.

## 0.12.x

- Added auto interface scaling based on current screen resolution.
- Added row-background progress display in the file list.
- Moved overall progress bar above Conversion settings.

## 0.11.x

- Added elapsed time and window title update every second during conversion.
- Added pre-stable OldWin branding.
